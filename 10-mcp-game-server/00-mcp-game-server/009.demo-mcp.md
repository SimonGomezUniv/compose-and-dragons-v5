# 📡 Docker MCP Gateway + Function Calling
> 👀 `compose.yml`
```bash
docker compose up -d
```
> Open **Inspector**

```bash
# Transport Type: Streamable HTTP
# URL: http://mcp-gateway:9011/mcp
# Connection Type: Via Proxy
# Use the provided URL in the logs to connect to the MCP Inspector
```
> Connect with **Golang MCP Client**

```bash
docker compose exec talk-to-dungeon /bin/sh
```
