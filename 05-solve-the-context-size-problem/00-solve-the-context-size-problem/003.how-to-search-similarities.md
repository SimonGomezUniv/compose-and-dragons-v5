# How to search similarities?
> We will use Embeddings 🤔 | transform `text` into `numbers`

- An **embedding** (or vector) = <span class="dodgerblue">**mathematical**</span> representation of the semantic value of a text
- Giving **"coordinates"** to data in a mathematical space where:
  - **Similar** elements are <span class="forestgreen">**close**</span>
  - **Different** elements are <span class="indianred">**distant**</span>
