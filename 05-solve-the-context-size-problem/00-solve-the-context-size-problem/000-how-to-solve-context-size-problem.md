# How to solve the context size problem

- Limit the number of messages stored in memory
- Use <span class="dodgerblue">**RAG**</span> (Retrieval-Augmented Generation) techniques
- Summarize long conversations periodically (<span class="dodgerblue">**Context "Packing"**</span>)
- Change the default settings of the LLM Engine (we'll see this later)
