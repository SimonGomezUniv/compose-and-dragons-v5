# RAG: retrieval augmented generation

- RAG will help to make the context 
  - <span class="dodgerblue">**Smaller**</span> and 
  - More <span class="dodgerblue">**relevant**</span>
- By <span class="dodgerblue">**Breaking**</span> information into <span class="dodgerblue">**chunks**</span>
- And by <span class="dodgerblue">**Searching similarities**</span> between the <span class="dodgerblue">**chunks**</span> and the <span class="dodgerblue">**user question**</span>
  - Then, we won't exceed the <span class="dodgerblue">**context limit**</span>
  - SLM will be more <span class="dodgerblue">**focused**</span>
  - SLM will be <span class="indianred">**faster**</span> 🚀
