# 🏰 🐙🐲 Compose & Dragons

## 📝 Important Notes

- Check the names of the NPCs in the metadata file `dungeon-mcp-server/data/dungeon_metadata.json` and in the sheet files in `non-player-characters/sheets/`. They should match.
- In the sheet files in `non-player-characters/sheets/` there is a new field `secretWord`.
- The fields of the JSON files are used to fill in the instruction template in `/docs/dnd.chat.system.instructions.md`.
