# Chat with a NPC

## If the NPC sheet does not exist
- Create it by generating a new NPC (see previous section)
- Generate a JSON file to identify the NPC
## Else, enter in chat mode
- Load the NPC character sheet (narrative format)
- Start a conversation with the NPC
- Use the NPC's personality, background, and traits to guide the conversation
